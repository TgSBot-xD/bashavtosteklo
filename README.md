# БашАвтоСтекло

Современный лендинг для сервиса по ремонту и замене автостёкол в Уфе. Переработка устаревшего сайта [башавтостекло.рф](http://башавтостекло.рф) с использованием современного стека технологий.

## О проекте

Проект представляет собой одностраничный сайт (лендинг) для ИП, специализирующегося на услугах:

- Замена автомобильных стёкол (лобовое, боковое, заднее)
- Ремонт сколов и трещин на автостёклах
- Тонирование стекол автомобиля
- Тонирование и бронирование оптики/фар
- Оклейка автомобиля защитной плёнкой
- Обслуживание автомобильных кондиционеров

## Технологический стек

### Frontend

- **Next.js 16.1.1** - React-фреймворк с поддержкой SSR/SSG
- **React 19.2.3** - библиотека для построения UI
- **TypeScript 5.9.3** - типизированный JavaScript
- **Tailwind CSS 4.1.18** - utility-first CSS фреймворк
- **Radix UI** - доступные UI-компоненты (Dialog, Navigation Menu, Hover Card)
- **Lucide React** - иконки
- **next-themes** - поддержка светлой/тёмной темы

### Инструменты разработки

- **pnpm** - пакетный менеджер
- **ESLint** - линтер кода
- **Prettier** - форматирование кода
- **Jest** - юнит-тестирование
- **Playwright** - E2E и визуальное тестирование
- **Steiger** - валидация FSD архитектуры

### Архитектура

Проект следует методологии **Feature-Sliced Design (FSD)**:

```
src/
├── app/           # Конфигурация приложения
├── pages/         # Страницы и композиция
│   └── landing/   # Главная страница лендинга
│       ├── config/     # Данные для секций
│       ├── models/     # Типы и модели
│       ├── tests/      # Тесты (unit + visual)
│       └── ui/         # UI компоненты страницы
└── shared/        # Переиспользуемые ресурсы
    ├── config/    # Общие конфигурации
    ├── lib/       # Утилиты
    ├── models/    # Общие типы
    └── ui/        # UI-kit компоненты
```

## Установка и запуск

### Требования

- Node.js 20+
- pnpm 10.28.0

### Установка зависимостей

```bash
pnpm install
```

### Доступные команды

```bash
# Разработка
pnpm dev              # Запуск dev-сервера (http://localhost:3000)

# Сборка
pnpm build            # Production сборка
pnpm start            # Запуск production сервера

# Качество кода
pnpm lint             # Проверка ESLint
pnpm lint:fix         # Автоисправление ESLint
pnpm format           # Форматирование Prettier
pnpm FSD              # Проверка FSD архитектуры
pnpm FSD:fix          # Автоисправление FSD

# Тестирование
pnpm test             # Запуск Jest тестов
pnpm test:up          # Обновить снапшоты Jest
pnpm test:pw          # Запуск Playwright тестов
pnpm test:pw:up       # Обновить визуальные снапшоты

# Все проверки
pnpm all              # FSD + format + lint + test + test:pw
```

## Структура лендинга

Лендинг состоит из следующих секций:

1. **Header** - шапка с логотипом, навигацией и контактами
2. **Hero Section** - главный экран с призывом к действию
3. **Services** - карточки услуг (6 основных услуг)
4. **Why We** - преимущества работы с сервисом
5. **Examples Work** - примеры выполненных работ
6. **Reviews** - отзывы клиентов

## Особенности реализации

### Адаптивность

- Desktop-first подход
- Полная адаптация под мобильные устройства
- Мобильное навигационное меню
- Оптимизированные карточки для разных разрешений

### Темы

- Светлая и тёмная тема
- Автоматическое определение системных настроек
- Сохранение выбора пользователя

### Тестирование

- **Unit-тесты** (Jest) - для бизнес-логики и компонентов
- **Визуальные тесты** (Playwright) - скриншотное тестирование на:
  - Chromium (Desktop)
  - Firefox (Desktop)
  - WebKit (Desktop)
  - Mobile Chrome
  - Mobile Safari
  - Светлая и тёмная темы

### Производительность

- Server-Side Rendering (SSR) через Next.js
- Оптимизация изображений
- Code splitting
- Tree shaking
