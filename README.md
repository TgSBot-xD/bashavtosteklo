# БашАвтоСтекло — UI прототип лендинга (интерактивный)

Интерактивный прототип одностраничного сайта-визитки для сервиса автостёкол (замена/ремонт/тонировка/плёнки/оптика/кондиционер).  
Стилистика: минимализм, крупная типографика, “дорогие” отступы, аккуратные микровзаимодействия.

## Что это и зачем

- Проверить структуру лендинга, логику CTA, адаптивность, модалки “Подробнее”.
- Упростить обсуждение дизайна до разработки (что оставить / что убрать).

## Возможности прототипа

- Режим предпросмотра **Desktop / Mobile** (переключатель справа сверху меняет ширину “рамки”).
- Кликабельные карточки:
  - **Услуги** → открывается модалка “Подробнее”
  - **Примеры работ** → модалка кейса
- На узкой ширине появляется **нижняя панель быстрых действий** (звонок / WA / маршрут / запись).
- Плавный скролл по якорям.
